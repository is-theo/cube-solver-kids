# 기여 가이드 (Contributing Guide)

이 프로젝트는 아이들을 위한 큐브 솔버입니다. 코드의 안정성과 가독성을 중요하게 생각합니다.

## 🛠 개발 환경 설정

1. 저장소를 클론합니다.
2. `npm install` 로 의존성을 설치합니다.
3. `npm run dev` 로 개발 서버를 실행합니다.

## 🧪 테스트 실행

새로운 기능을 추가하거나 버그를 수정할 때는 반드시 테스트를 실행하여 기존 기능이 망가지지 않았는지 확인해주세요.

```bash
npm test
```

테스트 코드는 `src/lib/*.test.ts` 파일에 작성되어 있습니다. 새로운 로직을 추가할 경우 해당 테스트 파일에 테스트 케이스를 추가해주세요.

## 📁 코딩 컨벤션

- **TypeScript**를 엄격하게 사용합니다. `any` 사용을 지양해주세요.
- **컴포넌트**는 `src/components` 에 위치하며, UI 로직과 비즈니스 로직을 분리하려고 노력합니다.
- **핵심 알고리즘**은 `src/lib` 에 위치하며, 순수 함수로 작성하여 테스트가 용이하도록 합니다.

## 🚀 배포

`npm run build` 명령어로 빌드할 수 있습니다. `dist` 폴더에 결과물이 생성됩니다.
